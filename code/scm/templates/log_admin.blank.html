  1. Best Window size is 1024x768(big font). Table's title is on the top of frame (Dark Blue Area)。
  2. Please use the button on top of frame to search working log.
  3. All List is order by Date/Time at least.
  4. [ List All Record ] will list all record by date/time.
  5. [ Delete All Record ] will Delete all record. Please make sure before click this button.
  6. Search / listing work log record ;choice one of rule [user name, work id or point date(before the point date)] and click its button to search data.
  7. On search list, there are 30 record per page. Click arabic numeral to change 1~10 page and click [per-page] / [next page] to jump 10 page.
  8. On search list, using click box(on the front of list) to choice records and click [Delete choice record] can delete records that be choiced.
  9. Delete needless log record can be more efficacy and free more space in system.
